Instructions for Corpus Analyzer

Steps for using the corpus analyzer tool:

1) Click the 'Corpus Analyzer' option on the left sidebar.

2) Give a title to the analysis.

3) Log In to the PoolParty server and open the desired project to analyze.

4) Copy the Project ID number found under the project's name and paste in the project ID field.

5) Copy the Corpus ID number found under the corpus's name in the Corpus Management tab, and paste in the corpus ID field.

6) In the PoolParty server project's webpage, click the export button found in the 'PROJECT' tab.

7) Click on 'RDF Project Export', select format 'RDF/XML', and click 'Export' to download a file with extension 'rdf'.

8) In the corpus analyzer, upload the just recently file by clicking the 'Choose File' button.

9) Click upload to analyze the corpus.


Instructions for Entity Extractor

Steps for using the entity extractor tool :

1) Type of paste the desired text to analyze.

2) Choose the desired thesaurus that will analyze the text.

3) Click analyze.


If it is necessary to upload a new thesaurus that is not on the list, follow the instructions:

1) Click the upload thesaurus option in the 'Entity Extractor' tool.

2) In the title field, give a descriptive title to the thesaurus that will be uploaded.

3) Log In to the PoolParty server and open the desired project to analyze.

4) Copy the Project ID number found under the project's name and paste in the project ID field.

5) In the PoolParty server project's webpage, click the export button found in the 'PROJECT' tab.

6) Click on 'RDF Project Export', select format 'RDF/JSON', and click 'Export' to download a file with extension 'rj'.

7) Back to the 'Entity Extractor' tool, click 'Choose File' button and upload the recently downloaded file.

8) Click 'Upload' button.

Notes: To delete the thesaurus that are already in the list, click on the delete list link, found in the 'Entity Extractor' page

To update a specific thesaurus, click on the 'upload new thesaurus' link and provide the same name used on the existent thesaurus along with the same project ID.
